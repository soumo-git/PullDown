// features

Everything you need.
Nothing you don't.

Built on the most battle-tested download and conversion engines available. Just with an interface that doesn't suck.

Full Quality Downloads

4K, 8K, VP9, AV1, H.265 — no artificial caps. Video and audio fetched separately and muxed via ffmpeg for maximum quality.

Video Converter

Convert any local file between any format. mp4, mkv, mp3, flac, gif — if ffmpeg supports it, PullDown can do it. 100% offline.

In-App Browser

Browse YouTube, Vimeo, or any platform inside the app. A "Download This" button appears on any video page — one click to queue it.

Swappable Engines

yt-dlp and ffmpeg ship as replaceable sidecars. When updates drop, you're notified — you stay in control, no silent replacements.

Parallel Queue

Download multiple videos simultaneously. Pause, resume, or cancel individual items without affecting the rest of the queue.

Private by Design

No telemetry, no analytics, no cloud. Everything runs locally. Open source — read every line of code yourself on GitHub.
